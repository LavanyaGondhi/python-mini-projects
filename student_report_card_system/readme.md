# Student Report Card System

This is a Python mini project built while learning Python functions.

## Description

This program generates a simple report card for a student.
It stores student details and subject marks in a dictionary and
calculates grades, average marks, and final result.

The program demonstrates how functions can be used to organize
code and perform specific tasks.

## Features

- Calculate grades based on marks
- Calculate average marks
- Check pass or fail condition
- Display a formatted report card
- Use multiple functions for modular design

## Concepts Used

- Python Functions
- Dictionaries
- Loops
- Conditional Statements
- List conversion
- Formatted printing

## How to Run
Run the program using:
         python student_report_card.py
