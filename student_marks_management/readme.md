# Student Marks Management System

This is my first Python mini project.

## Project Description
This is a console-based Python program that manages student marks.
It calculates total marks, average marks, pass/fail count, and topper.

## Concepts Used
- Python variables
- Lists and nested lists
- for loops
- if-else conditions
- Basic logic building

## Purpose
I created this project while learning Python basics to improve my logic
and understand how lists and loops work in real examples.
